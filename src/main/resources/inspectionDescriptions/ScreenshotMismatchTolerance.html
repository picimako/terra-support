This inspection reports various problems regarding the misMatchTolerance property that may be defined for screenshot validation related functions such as:
  • Terra.validates.element
  • Terra.validates.screenshot
  • Terra.it.validatesElement
  • Terra.it.matchesScreenshot

The following validations are in place at the moment that check whether the specified values are:

  • outside of the allowed boundaries, as in values not between 0 and 100,
  • not numeric values, such as object literals, booleans, Strings, etc.
  • are above a preset threshold, for which the default value is 0.5.

Below, on the Options panel you can override the threshold value.

For further details please consult the Terra Webdriver.io Utility Developer's Guide.

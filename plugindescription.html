Provides support for working with Cerner's various Terra libraries and components.

This plugin focuses mostly on the test automation aspect of the Terra libraries but has some support for Terra UI development as well.


Terra-toolkit wdio utility

This part of the plugin provides the following features for easier handling of screenshots, and for reporting coding issues:

  • inspections for viewport and screenshot handling
  • tool window/panel for working with Terra wdio screenshots much easier, extended with bulk operations on them
  • added and extended Quick Documentation popup contents
  • custom editors for previewing diff images, and comparing reference and latest screenshots


Terra UI components

Adds extended Quick Documentation popup content for React tags imported from terra packages. For details on each functionality head to the readme file in the plugin's GitHub repository.
